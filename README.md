# Recipe Box
As a user Recipe Box let's you store, add, edit, and delete recipes. AAU I should be to add a recipe title, image link, description, prep time, ingredients, and instructions for each recipe.

## Technology Used
* Node.js
* Express
* MongoDB
* HTML
* CSS3
* Google Fonts
* Colors.co
